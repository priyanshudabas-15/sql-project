#data analysis-project
coffee shop sales
This project presents an end-to-end data analytics solution for a coffee shop using SQL for data analysis and Power BI for interactive dashboard visualization.

The objective was to transform raw transactional data into actionable business insights that can support operational and strategic decision-making.
1️ SQL-Based Sales Analysis
In this phase, SQL was used to clean, transform, and analyze transactional-level data.
Key Analysis Performed:
1 Total Revenue Calculation
2 Revenue by Product Category
3 Revenue Contribution % using Window Functions
4 Hourly Sales Trend Analysis
5 Order Volume Analysis
6 KPI Generation (Revenue, Orders, Category Share)

SQL Concepts Used:GROUP BY,SUM(),COUNT(),ROUND(),HOUR(),DATE_FORMAT(),Window Functions (OVER())


2️ Power BI Interactive Dashboard
The cleaned and analyzed dataset was then visualized in Power BI to create an executive-level dashboard.
Dashboard Features:
1 KPI Cards:Total Revenue,Total Orders,Revenue % by Category
2 Hourly Sales Trend Chart
3 Category-wise Revenue Distribution
4 Interactive Filters (Date / Category)
5 Clean and professional UI design

Business Value Delivered:
-Identified peak sales hours for staffing optimization
-Highlighted top-performing product categories
-Revealed revenue concentration patterns
-Enabled quick executive-level performance monitoring
